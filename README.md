# 創作居酒屋「粋 -IKI-」ランディングページ

おしゃれな創作居酒屋のランディングページです。

## 特徴

- **モダンなデザイン**: 和の要素とモダンなUIを融合
- **レスポンシブ対応**: スマートフォン、タブレット、PCに完全対応
- **スムーズなアニメーション**: スクロールに応じた美しいアニメーション
- **使いやすいナビゲーション**: 直感的なメニュー構成

## ファイル構成

```
├── index.html      # メインHTMLファイル
├── styles.css      # スタイルシート
├── script.js       # JavaScript（インタラクション）
└── README.md       # このファイル
```

## セクション

1. **ヒーロー**: 印象的なファーストビュー
2. **コンセプト**: 店舗のコンセプト紹介
3. **メニュー**: 代表的なメニューの紹介
4. **店内**: 店内の雰囲気やシート紹介
5. **アクセス**: 店舗情報とアクセス方法
6. **予約**: 予約方法のご案内

## 使い方

1. ブラウザで `index.html` を開く
2. すべてのファイルが同じディレクトリにあることを確認

## カスタマイズ

- **色**: `styles.css` の `:root` セクションでカラー変数を変更
- **フォント**: Google Fontsの「Noto Sans JP」「Noto Serif JP」を使用
- **画像**: SVGプレースホルダーを実際の画像に置き換え可能
